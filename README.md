# LaundryWala 🧺

An Android app for booking laundry pickups and tracking orders. 
Built as part of my Mobile Systems course at Hochschule Schmalkalden.

The goal was to design and build a complete, usable mobile workflow 
from scratch — not just a screen or two, but something that actually 
feels like a real app end to end.

---

## Features

- User login and registration
- Dashboard with active order overview
- Laundry booking with pickup scheduling
- GPS-based current location detection
- Photo capture, preview and deletion
- Real-time order tracking
- Order history and management

## How to run it

1. Clone this repo
```bash
git clone https://github.com/Purvika18/laundrywala-android-app
```
2. Open in **Android Studio**
3. Let Gradle sync finish
4. Run on emulator or physical device (Android 8.0+)
